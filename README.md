# SOC Incident Response Playbooks

A collection of practical Security Operations Center incident response playbooks designed to help analysts respond to common cybersecurity incidents in a structured and professional way.

## Playbooks Included

- Phishing
- Ransomware
- Malware
- Insider Threat
- Data Exfiltration
- Account Compromise

## Each Playbook Covers

- Detection
- Analysis
- Containment
- Eradication
- Recovery
- Lessons Learned

## Frameworks Used

- NIST Incident Response Lifecycle
- MITRE ATT&CK
- Security Operations Center best practices

## Skills Demonstrated

- Incident Response
- Phishing Investigation
- Malware Response
- Ransomware Response
- Threat Analysis
- Digital Forensics Basics
- IOC Collection
- MITRE ATT&CK Mapping
- Security Documentation
- Executive Communication

## Purpose

This repository is intended for educational and portfolio purposes. It demonstrates structured incident response thinking, documentation ability, and practical security operations knowledge.
